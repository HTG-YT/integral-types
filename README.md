# number-types
A Python Third-Party Library for various numeric types.

## Installation
Run `pip install number-types` to install this library.

## Usage
To create a variable with the types provided in the library, do this:
```python
from number-types import *

byte = Byte(1)
ubyte = UByte(1)
short = Short(1)
ushort = UShort(1)
long = Long(1)
ulong = ULong(1)
```
